# HomeAssistant教程   

## 简介

​	对于智能家居来说,现在的市场还是比较匮乏,目前接入较好的就是小米,常见的智能家居管家例如HomeKit,却只支持ios系统,对于安卓系统,HomeAssistant作为目前优秀的开源智能家居平台,由全球bug修复者和 DIY 爱好者社区提供动力,提供了多种预留接口,支持自定义开发,AI的蓬勃发展,让智能家居不再局限于过去的科技水平,如今的品牌壁垒,.
